# نظام RAG محلي بالعربية (PoC)

نظام أسئلة وأجوبة على مستنداتك يعمل محليًا 100% بدون إنترنت: ChromaDB لقاعدة المتجهات، bge-m3 للتضمين (1024 بُعدًا، دعم عربي قوي)، وqwen3.5:9b للإجابة — كلها عبر Ollama.

## التشغيل

```bash
bash run_poc.sh      # أول مرة: يتحقق من Ollama ويسحب bge-m3 ويفهرس ويشغّل الواجهة
bash run_ingest.sh   # فهرسة ملفات documents/ (أو أعد فهرستها بعد أي تغيير)
bash run_app.sh      # واجهة الويب — تفتح تلقائيًا على http://localhost:8765
```

الطرفية بدل الواجهة: `.venv/bin/python3 query.py` (تفاعلي) أو `.venv/bin/python3 query.py "سؤالك"`.

## الملفات

| الملف | الوظيفة |
|---|---|
| `documents/` | ملفاتك المصدر (pdf, docx, txt, md) |
| `chroma_db/` | قاعدة المتجهات (تُنشأ تلقائيًا) |
| `ingest.py` | استخراج (PyMuPDF للعربية) → تقطيع بحدود الجمل مع عزل الأقسام وحماية القوائم المرقمة وتتبع الصفحات → تضمين → تخزين، مع تنظيف تلقائي للملفات المحذوفة |
| `query.py` | نواة الاسترجاع والإجابة (تستخدمها الواجهة والطرفية معًا) |
| `app.py` + `ui.html` | واجهة ويب: مراحل حية، عداد وقت، بث الجواب، درجات مطابقة، رفع ملفات بفهرسة فورية |

## كيف يجيب النظام

1. **توجيه**: تحية → رد فوري؛ سؤال عن النظام → دردشة؛ أي شيء آخر → بحث في المستندات.
2. **استرجاع ذكي**: درجة مطابقة لكل مقطع، تعزيز للملف الذي يَرِد اسمه في السؤال وللقسم الذي يطابق عنوانه، واختيار أقل عدد ملفات يكفي.
3. **جلب القسم كاملًا**: سؤال عن سياسة/قسم بعينه → كل مقاطع القسم بترتيب القراءة + حقن «مراسٍ حتمية» مستخرجة آليًا (أسماء المبادئ، النطاق، المدد) يلتزم النموذج بنسخها حرفيًا.
4. **صدق المصادر**: يجيب من المستندات مع المصدر والصفحات؛ وإن لم يجد يصرّح بذلك ثم يقدّم «اقتراح عام (ليس من مستنداتك)» مفصولًا.

## مستويات التفكير

| المستوى | المقاطع | السلوك |
|---|---|---|
| 🚀 سريع | 6 | جواب مختصر جدًا |
| ⚖️ متوازن | 8 | موجز وافٍ |
| 🔬 عميق | 12 | مفصّل منظم |
| 🎯 دقة قصوى | 12 | مسودة ثم **تدقيق ضد النص الأصلي** — أعلى أمانة، الأبطأ |

من الواجهة: المحدد أعلى حقل الكتابة. من الطرفية: `عمق سريع/متوازن/عميق/دقيق` أثناء الحوار.

## إعدادات قابلة للضبط (متغيرات بيئة)

`RAG_LLM_MODEL` (افتراضي qwen3.5:9b — بدّله لنموذج أضخم مثل qwen3:14b)، `RAG_EMBED_MODEL`، `RAG_MAX_DISTANCE` (حد الصلة 0.75)، `RAG_CHUNK_WORDS` (300)، `RAG_OVERLAP_WORDS` (75)، `RAG_FILE_MARGIN`، `RAG_NAME_BOOST`، `RAG_HEAD_BOOST`، `RAG_DEPTH`.
